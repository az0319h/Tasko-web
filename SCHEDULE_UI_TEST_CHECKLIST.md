# 일정관리 페이지 UI 테스트 체크리스트

## 테스트 항목

### ✅ 1. 상태 아이콘 표시 확인
- [ ] 상태 아이콘이 지시사항과 함께 표시되는지 확인
- [ ] TaskStatusBadge와 동일한 아이콘 및 색상 사용 확인
- [ ] 아이콘 색상 확인:
  - ASSIGNED: 기본색 (FileText 아이콘)
  - IN_PROGRESS: text-yellow-500 (Loader2 아이콘)
  - WAITING_CONFIRM: text-red-500 (Clock 아이콘)
  - APPROVED: text-green-500 (CheckCircle2 아이콘)
  - REJECTED: text-red-500, opacity-50 (XCircle 아이콘)

**확인 방법**:
1. 일정관리 페이지 접속
2. 캘린더에서 Task 이벤트 확인
3. 각 Task의 지시사항 옆에 상태 아이콘이 표시되는지 확인
4. 아이콘 색상이 올바른지 확인

### ✅ 2. 카테고리별 색상 확인
- [ ] 카테고리별 색상이 올바르게 적용되는지 확인
- [ ] 색상 매핑 확인:
  - REVIEW (검토): 파란색 (#3b82f6)
  - REVISION (수정): 보라색 (#a855f7)
  - CONTRACT (계약): 초록색 (#22c55e)
  - SPECIFICATION (명세서): 주황색 (#f97316)
  - APPLICATION (출원): 청록색 (#14b8a6)

**확인 방법**:
1. 일정관리 페이지 접속
2. 캘린더에서 각 카테고리별 Task 확인
3. 이벤트 배경색이 카테고리에 맞게 표시되는지 확인

### ✅ 3. 관리자용 '내 일정' 선택 확인
- [ ] 관리자가 '내 일정' 선택 시 자신의 일정만 표시되는지 확인
- [ ] 드롭다운에서 '내 일정' 옵션이 표시되는지 확인
- [ ] 선택 시 자신의 일정만 캘린더에 표시되는지 확인

**확인 방법**:
1. 관리자 계정으로 로그인
2. 일정관리 페이지 접속
3. 드롭다운에서 '내 일정' 선택
4. 자신의 일정만 표시되는지 확인

### ✅ 4. 관리자용 '전체 사용자 일정' 선택 확인
- [ ] 관리자가 '전체 사용자 일정' 선택 시 각 사용자마다 별도의 캘린더가 표시되는지 확인
- [ ] 각 사용자 캘린더에 사용자명이 표시되는지 확인
- [ ] 각 사용자 캘린더가 세로로 배치되어 있는지 확인

**확인 방법**:
1. 관리자 계정으로 로그인
2. 일정관리 페이지 접속
3. 드롭다운에서 '전체 사용자 일정' 선택
4. 각 사용자마다 별도의 캘린더가 표시되는지 확인
5. 각 캘린더에 사용자명이 표시되는지 확인

### ✅ 5. 스크롤 확인
- [ ] 각 사용자 캘린더가 스크롤로 확인 가능한지 확인
- [ ] 페이지 전체가 스크롤 가능한지 확인

**확인 방법**:
1. '전체 사용자 일정' 모드에서 여러 사용자 캘린더 확인
2. 페이지를 스크롤하여 모든 캘린더를 확인할 수 있는지 테스트

### ✅ 6. 승인된 Task 종일 표시 확인
- [ ] 승인된 Task가 종일로 표시되는지 확인
- [ ] 승인된 Task가 캘린더에 표시되는지 확인 (이전에는 필터링되었음)

**확인 방법**:
1. Task를 승인 상태로 변경
2. 일정관리 페이지에서 해당 Task 확인
3. Task가 종일 일정으로 표시되는지 확인
4. Task가 캘린더에 표시되는지 확인

## 테스트 시나리오

### 시나리오 1: 기본 UI 확인
1. 일정관리 페이지 접속
2. 캘린더 뷰 확인 (월/주/일)
3. 상태 아이콘 및 카테고리 색상 확인
4. 이벤트 클릭 시 Task 상세 페이지 이동 확인

### 시나리오 2: 관리자 기능 확인
1. 관리자 계정으로 로그인
2. '내 일정' 모드 확인
3. '전체 사용자 일정' 모드 확인
4. 각 모드에서 일정 표시 확인

### 시나리오 3: Task 상태별 표시 확인
1. 다양한 상태의 Task 생성
2. 각 상태별 아이콘 확인
3. 승인된 Task의 종일 표시 확인

## 예상 결과

- ✅ 모든 상태 아이콘이 올바르게 표시됨
- ✅ 모든 카테고리 색상이 올바르게 적용됨
- ✅ 관리자 기능이 정상 작동함
- ✅ 승인된 Task가 종일로 표시됨
- ✅ UI가 반응형으로 작동함

## 문제 발생 시 확인 사항

1. **아이콘이 표시되지 않는 경우**:
   - 브라우저 콘솔에서 에러 확인
   - `getStatusIcon` 함수가 올바르게 import되었는지 확인
   - React 컴포넌트 렌더링 확인

2. **색상이 올바르게 적용되지 않는 경우**:
   - `getCategoryColor` 함수 확인
   - `convertToFullCalendarEvents` 함수에서 카테고리 색상 사용 확인

3. **전체 사용자 일정이 표시되지 않는 경우**:
   - `schedule-page.tsx`의 조건부 렌더링 확인
   - 사용자 목록 로드 확인
   - 각 사용자별 쿼리 실행 확인
